About Me

My name is Muhammed Kemal and I am a Software Developer from Turkiye.

As I have a great passion for game development, I work with various programming and scripting languages such as C++, C#, Python, and LUA; as well as different game engines including Unreal Engine, Unity, Godot, Source and Goldsource.

I released my first game Sullen: Light is Your Friend on IndieDB in 2014. Since then, I'm pouring my creativity on software and game development.

As a person who loves to learn, I'm always looking for ways to expand my knowledge and skills. I'm currently learning Web Development and Machine Learning to broaden my horizons.

I'm always open to new opportunities and collaborations. If you have a project in mind, feel free to contact me!

Take care!

Sullen: Light is Your Friend

This is the first game I published. I originally published it in 2014 on IndieDB. Later on, I gave it a remaster and re-released it on Steam in 2017

It's a linear horror game that runs on Unity and C#.

The Unfinished

This is a game I developed during the Covid-19 pandemic for a gamejam called "Celebrating Programmer Art"

The main idea behind that gamejam was to create a game with programmer art only. I used minimal assets from the engine itself and created the rest of the assets by myself (including the font!).

It uses Unreal Engine, Blueprints and C++ together.

C.A.P.I.T.A.L. (WIP.)

This is a work-in-progress story-based FPS game that I'm working on. It utilises Unreal Engine 5.

This linear story-driven game takes place in Ankara during a zombie outbreak. Utilises Blueprints and C++ together.

Liquor Wars: Tekel Defence
